Excel Processor

Generate PDFs Instructions:

  1. Upload your Excel file (.xlsx) containing the data
  2. Make sure your Excel file has an 'email' column if you want to send certificates via email
  3. Upload your HTML template
  4. Variables in the HTML template should match Excel column names exactly
  5. Choose your delivery method: ZIP download or Email
  6. Generate your PDFs
