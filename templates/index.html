Excel Processor

Generate PDFs Instructions:

  1. Upload your Excel file (.xlsx) containing the data
  2. Make sure your Excel file has an 'email' column for sending certificates
  3. Upload your HTML template
  4. Variables in the HTML template should match Excel column names exactly
  5. The system will generate PDFs and attempt to send them via email
  6. You'll receive a ZIP file containing all generated PDFs

View Fields Instructions:

Upload your Excel file to see the column names and first row data.
